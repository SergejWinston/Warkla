Публичная страница запуска проекта

Адрес страницы: /how_build. Эта страница доступна всем без авторизации и помогает быстро поднять проект локально или через Docker.

Быстрый локальный старт

Python 3.10+ Flask SQLite 
cd /home/sneroff/projects/Warkla
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py

Открой: http://localhost:5000

Docker запуск

1. Получить образ

docker pull ghcr.io/sergejwinston/warkla

2. Запустить контейнер

docker run -d \
  --name warkla \
  -p 5000:5000 \
  -e SECRET_KEY='change-me' \
  -e JWT_SECRET_KEY='change-me-too' \
  -e DATABASE_URL='sqlite:////app/data/warkla.db' \
  -e UPLOAD_FOLDER='/app/data/uploads' \
  -v $(pwd)/data:/app/data \
  ghcr.io/sergejwinston/warkla

Проверка после старта

  1. Открывается http://localhost:5000.
  2. Доступны страницы /login и /register.
  3. Регистрация и вход возвращают JWT.
  4. Транзакции создаются и отображаются.

Анимации и пиксели

Для приятного UX используем мягкие появления блоков и уважаем настройки доступности через prefers-reduced-motion.

  • Размерная сетка: шаг 4px/8px.
  • Иконки и картинки: 2x-ассеты для Retina.
  • Границы: 1px, без случайных дробных значений.
  • Проверка на ширинах: 360px, 768px, 1280px.
